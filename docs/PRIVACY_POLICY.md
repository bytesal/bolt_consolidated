# Bolt Engine Privacy Policy

## Data Collection

Bolt Engine may store limited Discord-related data required for moderation and server management features, including:

* User IDs
* Guild IDs
* Warning history
* Moderation case notes
* Ticket/modmail messages
* Staff activity logs
* Role persistence data

## Purpose of Data Storage

This data is stored solely to provide the bot’s moderation, logging, ticketing, and automation systems.

## Data Sharing

Bolt Engine does not sell, trade, or share collected data with third parties.

## Data Removal

Server owners or affected users may request data deletion by contacting the support server administrators.

## Security

All sensitive credentials are stored securely using environment variables. No private credentials are exposed publicly.

## Changes

This policy may be updated in the future as the bot evolves.
